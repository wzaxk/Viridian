# 🌙 Viridian

A modern and lightweight UI library for Roblox designed for speed, flexibility, and clean design.  
It provides a simple API for creating windows, tabs, sections, buttons, toggles, sliders, dropdowns and more.

---

## ✨ Features

- ⚡ Fast and lightweight
- 🎨 Theme support
- ✨ Smooth animations
- 🧩 Easy to use API
- 📦 AutoSaves config

---
